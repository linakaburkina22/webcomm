# Formula 1 Paddock Couture

## Project Overview
This project is a web communication and design project that explores Formula 1 as a cultural and visual environment, focusing on fashion, identity, and editorial storytelling.

The website presents drivers not only as athletes, but as style icons, combining motorsport aesthetics with fashion-oriented communication.

---

## Technologies Used

- HTML5 (semantic structure)
- CSS3 (custom editorial styling)
- Bootstrap 5 (layout and carousel)
- JavaScript (interaction and cookies)
- Google Fonts (DM Serif Display & Montserrat)

---

## Key Features

- One-page website with clear section-based navigation  
- Responsive design using Bootstrap  
- Interactive carousel for driver presentation  
- Editorial-style layout with strong typography hierarchy  
- Cookie banner that stores user preferences  
- Theme toggle functionality
