Technical Presentation

Formula 1 Paddock Couture

A web communication project that presents Formula 1 as a fashion-oriented editorial platform through branding, image, and identity.

01

Project Idea

This project presents Formula 1 not only as a sport, but also as a visual and cultural environment shaped by style, branding, and personal identity.

The goal was to combine editorial aesthetics with web communication techniques studied during the course.

02

Technologies and Design

  • HTML5 and semantic sections
  • CSS3 with custom editorial styling
  • Bootstrap 5 for responsive layout and carousel
  • JavaScript for interaction and cookies
  • DM Serif Display and Montserrat for visual hierarchy

03

Interface and Navigation

These screenshots show the homepage hero section and the navigation menu. The visual design uses a strong editorial hierarchy, while the dropdown navigation improves access to the driver sections.

The interface combines a dramatic visual tone with a clear reading structure.

04

HTML Structure

This code shows the semantic organisation of the page with separate sections such as intro and drivers.

It demonstrates a clear one-page structure and readable markup.

05

Bootstrap Carousel

The project uses Bootstrap 5 to build a responsive carousel for the drivers section.

Custom CSS gives it a more editorial look and helps integrate it into the overall visual identity.

06

Profiles and Content Hierarchy

This part of the code shows how the driver profiles are structured with separate articles, images, and text blocks.

It supports stronger hierarchy and a magazine-inspired reading flow.

07

Cookies

The site includes a cookie banner that stores user preference after acceptance.

This directly applies a key topic of the course in a real interface and shows how the project uses JavaScript beyond static layout.

08

Lessons Learnt

This project helped develop skills in semantic HTML, custom CSS, Bootstrap, JavaScript, cookies, and visual hierarchy.

It also showed how design choices can influence the communication value of a website.

09

Long-Term Plans

The project can be expanded with more screenshots, additional content, and further refinement of the visual identity.

It could also become part of a wider personal web portfolio.
