# Formula 1 Paddock Couture

## Project Overview
This project explores Formula 1 as a visual and cultural space where fashion, identity, and branding intersect.  
The website is designed as an editorial experience inspired by fashion magazines.

## Structure
- index.html
- css/style.css
- js/script.js
- img/
- pitch/

## Technologies
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Cookies (user preferences and theme)

## Features
- Responsive layout (Bootstrap)
- Driver carousel
- Editorial sections (profiles, teams, trends)
- Theme switch (light/dark)
- Cookie-based preferences
- Embedded video

## Notes
- Carousel images are JPG
- Profile images are PNG
- All images should be placed in the `/img` folder
